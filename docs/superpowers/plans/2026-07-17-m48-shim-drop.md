# M4.8 ‚Äî `:Sentence` Shim Drop Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Retire the M4.5a transition shim: `:Fragment` becomes the only fragment label, the deprecated code aliases disappear, `SentenceRepository` finishes its rename, and a one-shot migration retargets live graphs.

**Architecture:** Pure code-surface + graph-label work. Wire format untouched: event type names, `AggregateType.SENTENCE`'s value `"Sentence"`, `Sentence-{uuid}` stream names, `sentence_id` payload/property names, and handler FILE names all stay. The alias flip is ATOMIC (aliases + call sites + test patch paths in one commit). The migration handles the one subtlety: fragment-embedding vector indexes are named `fragment_embedding_*` but built on the `:Sentence` label ‚Äî they must be dropped and recreated on `:Fragment` or the ask vector channel silently breaks.

**Tech Stack:** Python 3.11, neo4j async driver, pytest.

**Spec:** `docs/superpowers/specs/2026-07-17-m48-shim-drop-design.md`
